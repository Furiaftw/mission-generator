# Mission Generator

A web-based randomized mission briefing generator for Naruto-themed shinobi roleplay servers.

## Features

- 8 mission types: Sabotage, Assassination, Bodyguard, Defense, Recon, Theft, Target Acquisition, Caravan Escort
- Rank-scaled threat levels (E through S)
- Location-specific environments, weather, and targets
- Points of interest, special rules, and extra modifiers
- One-click copy for pasting into Discord or Gemini
